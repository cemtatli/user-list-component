## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone [https://github.com/cemtatli/subs-card-design
```

Proje dizinine gidin

```bash
  cd my-project
```

Gerekli paketleri yükleyin

```bash
  npm install
```

Sunucuyu çalıştırın

```bash
  npm run start
```
